== Description ==

Adds an option to the "Edit Page" screen, allowing the user to easily choose another page to edit without leaving the "Edit Page" screen.


== Installation ==

1. Upload this plugin to the `/wp-content/plugins/` directory and unzip it, or simply upload the zip file within your wordpress installation.

2. Activate the plugin through the 'Plugins' menu in WordPress
